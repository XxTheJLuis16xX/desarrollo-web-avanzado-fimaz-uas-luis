## Descripción

Sistema CRUD de Productos desarrollado en PHP aplicando:

- Programación Orientada a Objetos
- Arquitectura MVC
- Uso de namespaces
- Autoload con spl_autoload_register()
- Manejo de excepciones con try/catch
- Conexión PDO con MySQL

---

## Funcionalidades

- Registrar productos
- Listar productos
- Editar productos
- Eliminar productos
