# Práctica 1 – Parcial 2 Conexión PDO

## Objetivo
Se implementó una conexión a base de datos MySQL usando PDO en PHP, aplicando programación orientada a objetos y un CRUD básico de productos.

## Tecnologías utilizadas
- PHP 8+
- MySQL
- PDO
- POO
- Bootstrap

## Instrucciones de ejecución
1. Copiar la carpeta `practica-1` dentro de `htdocs`.
2. Iniciar Apache y MySQL en XAMPP.
3. Crear la base de datos `productos`.
4. Crear la tabla `producto`.
5. Abrir el navegador en `http://localhost:8080/practica-1`.

## Funcionalidades
- Registrar productos
- Listar productos
- Editar productos
- Eliminar productos
- Buscar productos por nombre o descripción
