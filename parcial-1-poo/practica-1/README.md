# Práctica 1 - Clases y Encapsulamiento en PHP

## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos en PHP mediante la creación de una clase con atributos privados, constructor y métodos de acceso.

## Descripción de la clase
Se creó la clase Usuario con los atributos privados nombre y correo. Además, se implementaron métodos getters y setters para acceder a la información de manera controlada.

## Instrucciones de ejecución
1. Iniciar Apache en XAMPP.
2. Abrir el navegador.
3. Entrar a la URL:
http://localhost:8080/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/index.php
